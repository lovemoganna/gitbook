Modules

模块化编程的一些基础概念.

What is a webpack module

相比较 NODEJS 模块,webpack 模块可以表示他们的依赖.

  • ES2015 import 声明
  • Commonjs require 声明
  • AMD define and require 声明
  • 在 css/sass/less file里面的 @import 声明
  • 样式表中的 image URL ,like url(...) 或者 HTML里面的 <img src…>= file.

Supported module types

  • TypeScript
  • ESNext
  • Sass
  • Less
  • Stylus
